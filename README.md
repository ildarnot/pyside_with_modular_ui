# pyside_with_modular_ui
В этом проекте мы реализуем подход к модульному ui в однооконном приложении. Проверим возможность раздельного редактирования окон приложения.
Также файлы из этого проекта можно брать для построения готовых минимальных ui окон с подвязанным функционалом 

Основной функционал импорта из одного ui в другой обеспечивается данными строчками:

        # Создаём экземпляр Selection_window (с UI и логикой)
        self.selection_widget = Selection_window()
        # Берём его центральный виджет для встраивания в stackedWidget
        selection_central_widget = self.selection_widget.centralWidget()
        # Добавляем в stackedWidget на страницу с индексом 1
        self.stackedWidget_2.insertWidget(1, selection_central_widget)
